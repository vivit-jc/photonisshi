# コンセプト
- 写真撮影により簡単に日誌を作るアプリ
- 写真を撮った時間帯により、自動で画像にタグを付与する（開店時、閉店時等）
- webアプリだが、スマホで使うことを想定している。写真撮影以外の機能はPCからでも使える。
- 本ソフトは試作であるため、本番時に必要な機能の一部はオミットする

# 環境
- vue3
- DBはsupabase、画像ストレージはsupabase storage
- ホスティングはnetlify
- スマホで起動した際、カメラと連動するようにする

# 仕様-アクセス時
- ページにアクセスした時、ユーザーはダイアログにユーザー名を入力しOKを押す。
- そのユーザーが既に存在すればそのユーザーのページを開き、そうでなければ新規ユーザーページに移動する
- ユーザー名入力欄は、一度入力した値を記憶しておき２回目以降はそのままOKを押すだけにする

# 仕様-ダッシュボード
- ダッシュボードには「カメラを起動」ボタンがあり、それを起動し撮影することで、その日の日誌の写真記録を追加する（複数枚可）
- 写真撮影の時間帯によってタグが自動的に付与される。タグはタグ設定画面で、「タグの名前」「時間帯（開始時間、終了時間」を設定することで追加できる。複数タグの時間帯が被っている場合は、警告を表示し、開始時間の早い方のタグとして扱う。
- 写真撮影の時間を記録しておき、タグの時間帯を後から変更するとその時間に追従してそのタグに変更される。
- ダッシュボードではコメントも記入できる。コメントはテキスト欄に入力しコメントボタンを押すと書き込まれる。コメントには書き込み時間も記録され、表示される。
- 写真やコメントは長押しで削除できる。削除する際は確認ダイアログを表示する。

# 仕様-画像一覧
- 「画像一覧」画面では、ユーザーは日にちごとの写真を閲覧できる。サムネイル表示に対して、クリックしたら拡大した画像が見れる。
- コメントも時系列順に画像に混ざって表示される。サムネイルは吹き出しの形で、クリックすると文章が表示される。
- 画像一覧画面では、画像は指定した月やタグでフィルタリングできる。
